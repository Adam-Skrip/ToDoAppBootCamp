Welcome to ToDoApp page!

ToDoApp is website where you can add tasks for shopping, school, work etc. You can group this tasks to list and move between them. These features are accessible by logged or registered users.

This web application is for Visma Hackathon BootCamp. Backend is in .NET #C and frontend in Angular. For Better work between BE and FE we use Swagger documentation. Also this web app is currently on Azure Cloud.
